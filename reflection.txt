(system_report.sh) reflection answers:

1- The pipe (|) sends the standard output of the command on its left into the standard input of the command on its right
(e.g., who | wc -l passes the who output into wc -l to count lines).

2- > redirects stdout to a file and truncates/creates the file (overwrite). >> redirects stdout and appends to the file (creates if missing).
